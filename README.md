# Y2JB2

# How to use
- Put [kexp.bin](https://github.com/ufm42/kexp) and [elfldr.elf](https://github.com/ps5-payload-dev/elfldr) into public folder.
- Update IPs in [splash.html](https://github.com/ufm42/Y2JB2/blob/main/splash.html) and [app.js](https://github.com/ufm42/Y2JB2/blob/main/public/app.js).
- Upload `splash.html` using ftp into `../download0/cache/splash_screen/aHR0cHM6Ly93d3cueW91dHViZS5jb20vdHY=`.
- Run `node server.js` command (require node.js + dependencies).
